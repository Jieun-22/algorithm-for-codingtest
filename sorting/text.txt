
선택 정렬 : 가장 작은 데이터를 선택해 앞으로 보내는 정렬 방법  
            - O(n*n)
         
        
삽입 정렬 : 데이터 리스트에서 적절한 위치를 찾은 뒤에, 그 위치에 삽입되는 정렬 방법
            - 데이터가 거의 정렬되어 있을 때 효율적
            - 일반적으로 O(n*n), 거의 정렬된 상태 최선의 경우 O(n)
            
퀵 정렬 : 기준을 설정한 다음 큰 수와 작은 수를 교환한 후 리스트를 반으로 나누는 방식으로 동작
          - 피벗 : 교환하기 위한 기준
          - 데이터가 이미 정렬되어 있는 경우 느리게 동작
          - O(NlogN)
          
계수 정렬 : 데이터가 모두 담길 수 있는 범위의 리스트를 생성한 후 인덱스에 해당하는 수가 몇번 등장하는지 카운트한 후
            인덱스 값을 배열값의 개수만큼 출력
           - 데이터의 크기 범위가 제한되어 정수 형태로 표현할 수 있을 때만 사용 가능
           - O(N+K)  N: 데이터의 개수, K: 데이터중 최대값으 크기
